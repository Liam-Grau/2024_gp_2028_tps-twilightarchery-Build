# Twilight Archery

## More information:
This project is an arena-based TPS against a boss made on Unreal Engine 5. It can be played alone or in pairs.\
Unreal Engine Version : 5.4.4\
Visual Studio Version : 2022\
Development Time : 4 weeks\
Year of Development : 2024\
Build Link : https://drive.google.com/drive/folders/1oaj-2y_y9IyzAbFbgDppfcCJ1ePvCmby?usp=sharing

## Features:
Field:
- Item spawner
- Destructible building
- Climbable zone

Boss:
- Specific attack based on distance from target
- Moving
- Several areas of varying degrees of damage
- Four different attacks (Beam, Missile, Fireball and Slap)
- Rage mode
- Improved attack in rage mode
- aggro the player who touches him most

Player:
- Moving
- Sprint / Dodge / Jump
- Advanced stamina system
- Pick up objets
- Drop objets
- Inventory
- Limited number of arrows (Quiver)
- Loaded Shot
- Revive Partner (two-player mode only)
- Five Arrow types (Normal, Fire, Electric, Sleep and Special)

### Controls :
| Keyboard Controls | Controller Controls | Action |
| :---: | :---: | :---: |
| Z | Left Stick | Forward |
| Q | Left Stick | Left |
| S | Left Stick | Backward |
| D | Left Stick | Right |
| MOUSE | Right Stick | Camera Rotation |
| Left Shift | R1 | Run |
| Left Click | R2 | Shoot |
| E | L1 | Drop Arrow |
| F | L2 | Drop Medkit |
| Up | Up D-Pad | Reduce Arrow Type |
| Down | Down D-Pad | Increase Arrow Type |
| A | Left D-Pad / Right D-Pad | Use Medkit |
| Left ALT | A | Dodge |
| Space | B | Jump |
| C | Y | Climb |
| R | X | Revive |
| Backspace | Right Special Button | Pause |
